普通人养龙虾必会的心法口诀：内网穿透

发表于 2026-04-03 分类于 开发技术 阅读次数： Valine：

师傅口诀点拨，徒弟反杀强敌

武侠电视剧里经常有这样的桥段：徒弟出门意外遇到了敌人，交手没多久，就被人打得口吐鲜血。

这时师傅及时赶到，大喊一声：“竟敢伤我徒儿！”

徒弟虚弱喊道：“师傅，快救我！”

师傅却没有上前动手，而是隔空喊话：“徒儿莫慌，还记得为师平日教你的那套内功心法吗？”

镜头切到全身是伤的徒弟，趁着打斗间隙艰难回头，对师傅点了点头。

“好，听我口诀！” 随后师傅盘腿坐下，在一旁念起口诀，气势十足。

徒弟的功力突然开始大增，没过多久，就把敌人打倒在地。

这种桥段看起来是编出来的，但其实它是真的。

在AI时代，很多不懂技术的人，突然也会写代码了，但是时不时会遇到一些解决不了的困难。而这个时候，如果有一个懂技术的师傅说几句口诀，也可以马上功力大增。

这篇文章我要说的口诀就是内网穿透。

ArkClaw 帮我做好了游戏，却玩不了

前阵子我尝试了火山方舟的ArkClaw，也就是火山方舟版本的OpenClaw。

我想到的第一个任务就是让它帮我做一个好玩的小游戏。

几分钟后，他就把游戏做出来，并且运行起来了，给了我一个网址，但是这个网址我打不开。当时他给我的回复类似下面这样。

我想知道小龙虾对非技术背景的外行人是否友好。所以 我假设自己是一个完全不懂技术的外行人，问他为什么打不开，他就开始给我一顿分析，给我说了各种技术术语，让我排查。我还是模仿技术小白的身份跟他交流了很久，然后他说的技术术语越来越多，但问题始终没有解决。

本来我只有一个疑问，就是为什么我打不开这个游戏。经过他这一顿分析并没有解决，现在好了，我有10个问题了。

当我跟他说了一句口诀，他马上就搞定了

为了测试它，我决定恢复自己本来的程序员身份。我跟他说，我访问不了这个游戏，你能不能帮我做一个内网穿透？

他尝试了多种内网穿透的工具，终于有一个工具（Servo）可以工作，给了我一个URL链接。我点开就可以玩游戏了。

当时的聊天截图我没存。后来为了再次确认，我重置了小龙虾，又复现了一遍类似的聊天，下面是截图（为了节约时间，我直接指定他用Servo去做内网穿透）。

这次测试又成功了，它在1分多钟之后就告诉我内网穿透实现了，给了我一个URL链接。

点开这个URL，确实可以玩他做的2048小游戏了。

如果你想持续做内网穿透

如果你是一个没有技术背景的用户，也遇到了这样的问题，那么你下次就可以试试我这个秘密口诀：内网穿透。

但是你按前面的方式做内网穿透，会发现几个小时后它就用不了了。为了让有需求的人能持续使用内网穿透，我还是得简单科普几个要点。

内网

首先，我之所以访问不了他那个地址，是因为我的手机和他运行游戏的电脑不在一个内网里。

用通俗但不严谨的说法，是因为我的手机和ArcClaw所在的电脑（准确说是服务器，和电脑差不多意思），连的不是同一个路由器。我手机连的是我家里的路由器，而ArcClaw连接在了火山方舟团队的机房里。

内网穿透

为了能访问他这个电脑上的小游戏网站，就需要做一个内网穿透。原理就是找个中间商，把他电脑上的网址绑定到一个公共的网址上。于是我访问这个网址，就可以打开他电脑上的东西了：

我的手机 -> 我家路由器 -> 互联网 -> 内网穿透中间商 -> 火山方舟机房路由器 -> ArcClaw所在电脑（服务器）

这里你会发现，我们需要一个中间商。而这个中间商，要花钱买服务器，还有网费、电费，所以肯定不会一直免费给大家用，通常免费试用不稳定且有各种限制。

这也是为什么，第一次小龙虾尝试了好几个内网穿透工具，最后只有Servo成功了。

如果你用Servo免费版做内网穿透，每次它都会生成一个随机的临时网址，且这个网址几小时后就失效了。

稳定的内网穿透

如果想要稳定、长期可用的内网穿透，我的建议是用Cloudflare，它的免费额度很多，个人使用完全足够，但是需要有一个自己的域名。

域名

怎么有一个自己的域名呢？我最推荐全球最大的域名网站GoDaddy，是一个国外网站。当然你也可以在国内的网站比如腾讯云、阿里云买域名。

如果一个域名被别人买下来，你就买不了了，除非你找到买了那个域名的人卖给你。比如OpenClaw这个产品，之前改名就有原因是域名被人抢先注册了。

还有一些公司名字已经想好了，但是发现域名被人注册了，甚至会不惜重金把域名从别人手里买过来，比如马斯克就花了500万美金买下了 x.com。

也有人靠批量注册可能有用的域名，期待哪一天有公司买他的域名的大赚一笔。但是好域名维护成本很高，也不知道哪天会有人要买。

因为域名一般是一年交一次钱，大部分普通域名一年就几十到几百人民币，而长度短、含义明确的好域名每年费用没有上限。

比如我现在的博客用的是 www.xiaoming.io 这个域名，一年要700多块钱。有点贵，但是说实话，这么通俗易懂又不长的域名竟然能被我买到，还挺幸运的。

具体操作

至于具体怎么买域名，以及怎么把域名配置到Cloudflare，怎么让OpenClaw使用Cloudflare做内网穿透，现在这种AI时代，随便找个AI一问就能解决了。

甚至你可以手机跟豆包视频通话，摄像头对着电脑屏幕，让它一步步指导你。我试过，还挺好用的。所以这些细节就不讲了。

重要提醒

最后还有两个重要提醒。

如果你做了内网穿透，就相当于搭建了个网站。

第一点，你能访问这个网址，别人也就能访问。如果里面有个人隐私等内容，一定要让小龙虾帮你给这个网站设置登录密码。

第二点，如果网站只是你个人或者几个熟人私下使用，一般是没问题的。但如果你想把它做成公开网站让很多人用，并且你的OpenClaw在大陆地区，按有关规定需要做ICP备案。如果没备案而之后被发现了，可能会有严重的后果，切记。
